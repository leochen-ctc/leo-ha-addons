## 🚀 Installation

To add this repository to your Home Assistant instance:

1. Copy the URL of this repository: `https://github.com/leochen-ctc/leo-ha-addons`
2. Navigate to **Settings** -> **Add-ons** -> **Add-on Store** in your Home Assistant.
3. Click the three-dot menu (top-right) and select **Repositories**.
4. Paste the URL and click **Add**.
5. You should now see the "LeoChen addons repository" in the store.
